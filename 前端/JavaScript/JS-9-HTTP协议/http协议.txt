一.HTTP是什么
    HTTP 代表超文本传输​​协议，用于通过 Internet 构建请求和响应。HTTP 要求数据通过网络从一个点传输到另一个点。

    资源传输使用 TCP（传输控制协议）进行。在查看此网页时，TCP 管理您的浏览器和服务器（在本例中为 codecademy.com）
    之间的通道。TCP 用于管理多种类型的互联网连接，其中一台计算机或设备想要向另一台计算机或设备发送内容。HTTP 是连
    接双方的设备必须遵循的命令语言才能进行通信。


二.HTTP和TCP：是如何工作的
    当您在浏览器中键入www.codecademy.com等地址时，会先通过DNS（Domain Name System）将域名解析为IP地址，然后命
    令它打开一个 TCP 通道，连接到响应该 URL 的服务器（或统一资源定位器，您可以在Wikipedia上阅读更多相关信息）。
    URL 就像您的家庭住址或电话号码，因为它描述了如何联系到您。

    在这种情况下，发出请求的您的计算机称为客户端。您请求的 URL 是属于服务器的地址。

    一旦建立了 TCP 连接，客户端就会向服务器发送一个 HTTP GET请求，以检索它应该显示的网页。服务器发送响应后，它
    关闭 TCP 连接。如果您再次在浏览器中打开该网站，或者如果您的浏览器自动向服务器请求某些内容，则会打开一个新连接，
    其过程与上述相同。GET 请求是客户端可以调用的一种 HTTP 方法。您可以在本文中了解更多关于其他常用方法
    （POST、PUT和DELETE）的信息。
